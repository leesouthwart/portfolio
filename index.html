Cook Book

Community Cook Book is a recipe website built with Python and Flask, and uses a MongoDB database to store recipes. Users can add, edit and delete recipes in the database, as well as browse the databases recipes with whatever filters they fancy. There is a summary of the database that uses Google Charts, which can also be filtered by users.
